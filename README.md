
# DHU-micro-PI-Dataset

**DHU-micro-PI** is a PNG dataset of SEM images of micro hollow fiber membranes for semantic segmentation.

## Dataset Structure

```
DHU-micro-PI-dataset/
├── images/
│   ├── image1.png
│   ├── image2.png
│   └── ...
├── masks/
│   ├── mask1.png
│   ├── mask2.png
│   └── ...
└── README.md
```
